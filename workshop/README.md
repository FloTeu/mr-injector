# Workshop

## Setup
- Navigate in the workshop folder
- Create virtual environment (e.g. with [uv](https://github.com/astral-sh/uv))
- Activate the venv
- Select python interpreter of this venv
- Install all dependencies of the workshop `pyproject.toml` (e.g. with `uv pip install -r pyproject.toml`)
- Select venv kernel for the notebook
- Run the notebook (when prompted, install the `ipykernel` package)
